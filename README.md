# Proof of Concept - utilizando: GraphQL, React JS, NodeJS e MongoDB

# Comandos utilizados & dependências

* yarn init -y
* yarn add graphql-yoga
* yarn add nodemon -D
* node src/server.js
* yarn add mongoose
* yarn add express
